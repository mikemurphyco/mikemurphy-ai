## A fresh start

This is my first post written in MDX — plain Markdown with superpowers.

No page builder. No database. Just a file.

## What's different

In WordPress, this post would live in a database and get fetched on every request.
In Astro, this file gets turned into a static HTML page at build time — faster, simpler, and I own every byte of it.
