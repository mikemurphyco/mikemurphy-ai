### 🧠 AI Term of The Week: Connectors

**Definition:**
Connectors are the tools and plugins that AI agents like Claude, Codex, and others use to reach outside the chat window and actually do things in your other apps.

**Real examples:**

- Claude reading your Gmail inbox directly and summarizing the last 10 threads

- Claude posting that message into the right Slack channel for you

**Why it matters now:**

- Connectors are what turn a chatbot into an agent

- Every major AI platform is racing to build connector ecosystems (MCP, ChatGPT plugins, Codex tools)

- AI stops being a window you visit and starts being a coworker

**What is Conductor. Build?**
Conductor is a Mac app that lets you run multiple Claude Code agents in parallel, each in its own isolated workspace, so you can ship more code without waiting on a single agent to finish.

**Mike’s Take:**
It’s a little confusing and overwhelming at first, but if you are running multiple AI Agents (Claude Code, Gemini, Codex, Ollama) this tool handles them all in one interface.

**What it does:**

- Run a team of coding agents on your Mac.

- Create parallel Codex + Claude Code agents in isolated workspaces.

- See at a glance what they're working on, then review and merge their changes.

- Gives each agent its own visual workspace Lets you orchestrate, pause, and review multiple coding tasks from a single dashboard
