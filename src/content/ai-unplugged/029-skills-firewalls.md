### 🗞️******AI News of The Week: Anthropic Announces Skills**

**What it is:**
Anthropic just launched [**Claude Agent Skills**](https://www.anthropic.com/news/skills?utm_source=aiunplug.io&utm_medium=referral&utm_campaign=issue-29-skills-firewalls). Skills are reusable knowledge bases that make Claude more effective at domain-specific work without you having to repeat yourself. Think of Claude Skills as **instruction manuals that you create for Claude to follow when working on specific types of tasks**.

**Hot Take (observation by Mike):**
I believe Claude Skills are powerful and useful, but they are confusing as heck to understand because they overlap with many of the other great Claude tools (MCP, Agents, Projects, ClaudeCode.md file, etc. When I figure out how to actually use them, I’ll let you know, but the internet is all abuzz about Claude Skills!

![](/assets/media/2025/10/issue-029-1.png)

Skills are reusable knowledge bases that make Claude more effective at domain-specific work without you having to repeat yourself.

**Example:**
When you repeatedly find yourself explaining the same context, preferences, or procedures to Claude (like table names, filtering rules, or workflow steps), you can capture that information once in a Skill. Then, whenever Claude needs to perform similar tasks, it can reference that Skill instead of you having to re-explain everything.

 The key components are:

- A SKILL.md file that contains the main instructions

- Optional reference files for additional information

- Metadata (name and description) that helps Claude decide when to use the Skill

### 🧠 **AI Term of The Week: Computer Vision**

 Computer Vision is how AI learns to “see” the world. It allows machines to interpret and process visual data like photos, videos, or live camera feeds.

**Explain It Like I’m 5 Definition:**
Computer vision is like giving eyes to a computer so it can understand what it sees in pictures or videos.

### 🧰 AI Tool of The Week: ChatGPT Atlas Browser

[ChatGPT Atlas by OpenAI](https://chatgpt.com/atlas?utm_source=aiunplug.io&utm_medium=referral&utm_campaign=issue-29-skills-firewalls) is a web browser & AI assistant that brings ChatGPT’s capabilities directly to your web experience, offering instant answers, smarter suggestions, and hands-on help with privacy controls you can manage.

**Key features:**

- **ChatGPT sidebar:** Summarizes content, compares products, or analyzes data from any site.

- **Memory management:** Lets users choose what ChatGPT remembers for relevant, contextual assistance.

- **Agent mode:** Interacts with websites and completes tasks, like researching and shopping (available for Plus, Pro, Business accounts).

- **On-the-spot assistance:** Highlights text for AI help in emails, docs, or calendar invites.

- **Privacy controls:** Decide which sites ChatGPT can access, clear browsing history, and use incognito mode.

- **Smarter search:** Supports searching across text, images, videos, and news.

- **Customizable experience:** Adjust browser settings, tabs, and appearance for a personalized interface.

**Mike’s Take:**
So far, I’m impressed. I really don’t need more browsers in my life, but I do believe this is how web browsers will be in the future and expect Google Chrome to be like this very soon. There will be no need for you to use a separate ChatGPT app as you have access to your entire ChatGPT history in Atlas. It’s basically a hybrid of ChatGPT + Web Browser. It’s fast and simple to use.
