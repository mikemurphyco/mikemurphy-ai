## Screenflow 10: How To Export Audio Only

**Description:  
**This tutorial will show you how to export the audio only from Screenflow 10. This is useful to process your audio tracks in a DAW (Audition, GarageBand, Logic) and then import the polished audio back into Screenflow or to export compressed audio files to share online

**Lossless-Audio Only:  
**Lossless is an uncompressed audio file type that is similar to a .WAV file. This file is recommended if you want to export your audio only from Screenflow and process it in Adobe Audition or similar and then bring it back into Screenflow. The file size is larger than uncompressed.

**AAC Audio Only:  
**AAC is a compressed audio file similar to an MP3 file type and can be used to share online. AAC is supported by most podcast hosts, but I do recommend using a tool such as Audition or [Auphonic.com](http://Auphonic.com) to customize the Bit Rate for podcasting (ex. 128 Kbps) and meeting broadcast loudness standards before uploading to your podcast hosting company.

**How To Export Audio Only  in Screenflow 10 (Lossless-Uncompressed)**

1.  Go to File Menu
2.  Click on Export
3.  Click on Manual
4.  Click on the Preset Drop-down menu
5.  Select ‘Lossless-Audio only’ for an uncompressed AIFF file
6.  Click OK
7.  Name File
8.  Choose Destination
9.  Click ‘Export’

**  
How To Export Audio Only  in Screenflow 10 (AAC - Compressed)**

1.  Go to File Menu
2.  Click on Export
3.  Click on Manual
4.  Click on the Preset Drop-down menu
5.  Select ‘AAC-Audio only’ for an uncompressed AIFF file
6.  Click OK
7.  Name File
8.  Choose Destination
9.  Click ‘Export’
