How To Create A RodeCaster Pro Multitrack Template in Adobe Audition

![](https://cdn-images-1.medium.com/max/800/1*2lnKQ1e6RK4kX_BC9l885Q.jpeg)

### Who Is This For?

Podcasters, live-streamers, audio editors, and content creators who use the RodeCaster Pro as an audio interface and Adobe Audition as a DAW (digital audio workstation) to record and edit audio. This tutorial is for you if you want to have a Multitrack template specifically made for the RodeCaster Pro to speed up your podcast + audio workflow.

### What You Will Learn

- This tutorial will show you how to create a fully customizable and reusable Adobe Audition Multitrack Template that includes a track for each input (channel) on the RodeCaster Pro (RCP).
- Each time you create a new Multitrack Session in Adobe Audition, you can select the RodeCaster Pro Template and like magic, a new Multitrack session will be created specifically for the RodeCaster Pro. Each track in the Multitrack session will correspond to the input channels on the RodeCaster Pro.
- Any audio going into the RodeCaster Pro can be recorded on a separate track in Adobe Audition.
- Any track effects (EQ, Dynamics, etc) can be saved in the Template.
- The channels or inputs on the RodeCaster Pro include Stereo Mix, 4 Mono Mic Channels, USB (computer), TRRS (Smartphone), Bluetooth, and Sounds so any audio coming into the RodeCaster Pro will get routed to its own track in Adobe Audition CC 2022.
- Set up once (fully customize it to your workflow) and then reuse the template for each recording session. 👍

### **Step 1: Enable Multitrack Recording on RCP**

1.  Tap the Gear Icon
2.  Tap ‘Advanced’
3.  Tap ‘Audio’
4.  Tap ‘Multitrack’
5.  Toggle on Multitrack \> USB

### **Step 2: Audio Hardware Preferences in Audition**

1.  Open Adobe Audition and go to Preferences (Audition \> Preferences)
2.  Click on ‘Audio Hardware’
3.  Set the Default Input to RodeCaster Pro Multichannel
4.  Tip: Set the Default Output to RodeCaster Pro Stereo to monitor sound from the RodeCaster Pro
5.  Click ‘OK’

### Step 3: Create a Multitrack Session in Audition

**Note:** You can fully customize the Multitrack Session and even add Track Effects (EQ, Dynamics, Hard Limiter, etc) and any changes you make can all be saved in the Multitrack Template.

1.  Open Adobe Audition
2.  Create A New Multitrack Session (Cmd or Ctrl+ N)
3.  Session Name: RCP
4.  Folder Location: Click ‘Browse’ to choose location
5.  Template : ‘None’
6.  Sample Rate: 48000
7.  Bit Depth: 32-bits
8.  Mix: Stereo
9.  Click ‘OK’
10. **Track 1:** Click on Heading of Track 1 and rename ’Stereo Mix’
11. **Track 1:** Click on the arrow next to Default Stereo Input and select ’Stereo…RodeCaster Multichannel Left’
12. **Track 1:** Click the ‘R’ to Arm for Record
13. **Tracks 2–6:** Click on Track Headings and rename Mic 1, Mic 2, Mic 3, Mic 4
14. **Tracks 2–6:** For the Default Inputs, select Mono \> Mic 1, Mic 2, Mic 3, Mic 4
15. **Tracks 2–6:** Click the ‘R’ to Arm for Record
16. **Tracks 7–9:** Click on Track Headings and rename USB, TRRS, Sounds
17. **Tracks 7–9:** For the Default Inputs, select Stereo\> USB, TRRS, Sounds
18. **Tracks 7–9:** Click the ‘R’ to Arm for Record
19. **Track 10:** This is the default ‘Mix’ Adobe Audition Master Track that will get saved in your RCP Multitrack Template

### Step 4: Save Multitrack Session As Template

1.  Go to top menu and select ‘File’
2.  Click on ‘Export’
3.  Click on ‘Session As Template’
4.  Rename RCP or RodeCaster Pro
5.  Click ‘OK’

![](https://cdn-images-1.medium.com/max/800/1*_p6CpZustF-qG156CtawAg.jpeg)

### Step 5: Open the RCP Multitrack Template

**Scenario:** You have a new podcast interview and want to make sure you capture all of the audio going through the RodeCaster Pro. No problem.

1.  Open Adobe Audition
2.  Create a New Multitrack Session (Cmd/Ctrl + N)
3.  Click on arrow next to Template
4.  Select RCP or RodeCaster Pro as template
5.  Click ‘OK’
6.  Press Record when ready!

### Appendix: Where are Multitrack Template Files Saved?

Multitrack Session Files (.sesx) that you Save As Templates are located in a folder on your hard drive that is located in the ‘Media & Disk Cache’ Preferences. This is useful to know if you ever want to share a Multitrack Template you created or to manually install templates such as the free RCP template I created for this tutorial.

1.  [Download](https://www.dropbox.com/s/a6ulj1zi7pp75qg/RCP.sesx?dl=1) the Free RCP Multitrack(.sesx) Template from Dropbox (optional)
2.  Open Adobe Audition and go to Preferences (top menu)
3.  Adobe Audition \> Preferences\> Media & Disk Cache
4.  Locate ‘Adobe Session Templates’ and click ‘Browse’ to open folder (this is the folder where all Multitrack Templates are saved)
5.  Copy + Paste the free download .sesx file in the ‘Adobe Session Templates’ folder
6.  Create New Multitrack Session (Cmd + N)
7.  Select ‘RCP’
