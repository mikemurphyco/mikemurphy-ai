## After Effects CC 2023: How To Restore Missing Animation User Presets Folder

**Solids Folder Basics**

**Description:  
**This tutorial will show you how to locate and restore missing animation user presets in the Effects & Presets panel of Adobe After Effects CC 2023 after upgrading from AE 2022 to AE 2023.

**Keyboard Shortcuts:  
**Quit After Effects (Cmd/Ctrl + Q)

**Adobe Help:  
**<https://helpx.adobe.com/after-effects/using/effects-animation-presets-overview.html>  

**USER ANIMATION PRESETS HARD DRIVE LOCATION:  
**

1.  My Documents\Adobe\After Effects CC (Windows)
2.  Documents/Adobe/After Effects CC (macOS)

**How To Restore Missing Animation User Presets in AE 2023 (Mac Users):**

1.  Open Finder
2.  Go to the ‘Documents’ Folder
3.  Twirl open Adobe
4.  Twirl open the previous version of AE
5.  Twirl open the ‘User Presets’ folder
6.  Copy custom animation presets (Cmd + C)
7.  Twirl open AE 2023 Folder
8.  Twirl open User Presets folder
9.  Paste (Cmd + V) custom animation presets
10. Quit Adobe After Effects CC 2023 (Cmd + Q)
11. Re-open
12. Go to the ‘Effects & Presets’ panel
13. Twirl open ‘Animation Presets’
14. Confirm User Presets folder has been restored.

**How To Restore Missing Animation User Presets in AE 2023 (Windows Users):**

1.  Open Explorer
2.  Go to ‘My Documents’ Folder
3.  Twirl open Adobe
4.  Twirl open previous version of AE
5.  Twirl open ‘User Presets’ folder
6.  Copy custom animation presets (Ctrl + C)
7.  Twirl open AE 2023 Folder
8.  Twirl open the User Presets folder
