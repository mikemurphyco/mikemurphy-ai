## After Effects CC 2022: How To Skew or Slant Text

**Solids Folder Basics**

**Description:  
**This tutorial will show you how to skew or slant text and other others in Adobe After Effects CC 2022 using the Transform Effect.

**Font Used in Tutorial: **  
Prohibition (Adobe Fonts)  
 https://fonts.adobe.com/fonts/prohibition

**Keyboard Shortcuts to Learn:**

**How To Skew or Slant Text in Adobe After Effects CC**

1.  Add Text to Composition (Cmd/Ctrl + T)
2.  Type out some text (the font I used is called Prohibition)
3.  Select Text Layer
4.  Go to Effects & Presets Panel (Window…Effects & Presets)
5.  Search for ‘Transform’
6.  Drag ‘Transform’ on Text Layer
7.  Go to ‘Effect Controls’ Panel
8.  Change the Skew value to -10

**How To Animate Skewed Text**

1.  Drag play-head to the beginning of the Timeline
2.  Go to Effect Controls
3.  Change Skew to zero
4.  Click on the stopwatch to add a keyframe
5.  Drag play-head to the 1-second mark
6.  Change the Skew value to -10
7.  Drag play-head another 1 second
8.  Change Skew Value to zero
9.  Select all keyframes
10. Tap F9 to Easy Ease
11. Select all 3 keyframes
12. Copy + Paste to extend the animation

**How To Add Transform/Skew Effect to Multiple Layers**

1.  Add new Adjustment Layer (Opt/Alt + Cmd/Ctrl + Y)
2.  Go to Effects & Presets Panel (Window…Effects & Presets)
3.  Search for ‘Transform’
4.  Drag ‘Transform’ on Adjustment Layer
5.  Go to ‘Effect Controls’ Panel
6.  Change the Skew value to -10

**How To Add Background Layer with Adjustment Layer on top  
**

1.  Select all layers below Adjustment Layer
2.  Right-click and select ‘Pre-compose’
3.  ‘Move all attributes into new composition’
4.  Click OK
5.  Drag background elements below the Pre-composition
