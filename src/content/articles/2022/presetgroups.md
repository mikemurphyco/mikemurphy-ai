**How To Create Preset Groups in Adobe Media Encoder (AME) 2023:**

1.  Go to the top menu and click Presets
2.  Click ‘Preset’
3.  Click ‘Create Group’ or use the keyboard shortcut Cmd/Ctrl + G
4.  Enter ‘Group Name’ and tap Enter/Return

**  
How To Add Presets to Group:**

1.  Select a User or System Preset
2.  Drag preset on top of Group

**  
How To Add Presets to Group (using Aliases):**

1.  Twirl open System Presets or Use Search Field
2.  Right-click on any preset
3.  Select ‘Create Alias’
4.  Locate Alias in ‘User Preset’ area
5.  Drop on top of the Group folder
