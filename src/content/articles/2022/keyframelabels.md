## After Effects CC 2022: How To Add Keyframe Color Labels

**Solids Folder Basics**

**Description:  
**This tutorial will show you how to create and add keyframe color labels and select groups in Adobe After Effects CC 2022. This is helpful for keeping timelines organized.

**Software Used:

**How To Add Keyframe Color Labels in AE CC 2022:**

1.  Select a keyframe
2.  Right-click or Ctrl + Click
3.  Select ‘Label’
4.  Choose Color
5.  Keyframe will change new color

**How To Add Color Labels to Multiple Keyframes:**

1.  Select a keyframe
2.  Press Shift Key and select additional keyframes
3.  Right-click or Ctrl + Click
4.  Select ‘Label’
5.  Choose Color
6.  All selected keyframes will change new color

**How To Select by Color Label Group: **

1.  Select a keyframe
2.  Right-click or Ctrl + Click
3.  Click on ‘Select Keyframe Color Label’
4.  Choose ‘On Selected Layers’ or ‘On All Layers’ to select all even if not visible 
5.  Choose ‘Visible Keyframes on Selected Layers’ or ‘Visible Keyframes on All Layers’ to select keyframes by color if they are visible on the Timeline

**How To Edit Color Labels: **

1.  Select a keyframe
2.  Right-click or Ctrl + Click
3.  Click on ‘Edit Label Colors’ or go to the top menu…
4.  Select ‘After Effects’…’ Preferences’….Labels
5.  Change Label Name and/or colors
