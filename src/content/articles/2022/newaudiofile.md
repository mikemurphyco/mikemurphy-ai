## Adobe Audition CC 2022: New Audio File Recording

**Description:  
**This tutorial will show you how to create a new audio file recording in Adobe Audition CC 2022 in the Waveform Editor. This tip is useful if you are new to Audition and want to record a solo podcast episode or voiceover for a video, etc.

**Keyboard Shortcuts: **

- Shift + Cmd/Ctrl + N (New Audio File)
- Cmd/Ctrl + S (Save New Audio File)
- Option/Alt + i (Meter Input Signal)
- Shift + Spacebar (Start Recording)
- Spacebar (Stop Recording + Playback Audio)

**Software Used:  
**Adobe Audition CC 2022 ([Click To Try or Buy](https://mikemurphy.co/adobe))

**How To Record A New Audio File Recording:**

1.  Open Adobe Audition
2.  Go to File…New Audio File (Shift + Cmd/Ctrl + N)
3.  Enter ‘Filename’
4.  Sample Rate: 48,000
5.  Channels: Mono
6.  Bit Depth: 32-bit float
7.  Click ‘OK'

**How To Save New Audio File Recording:**

1.  Go to File…Save (Cmd/Ctrl + S)
2.  Enter File Name
3.  Click ‘Browse’ and navigate to Project Folder
4.  Format: Wave PCM
5.  Sample Type: Default
6.  Format Settings: Default
7.  Click OK

**How To View Meter Input Signal (Audio Levels):**

1.  Go to View Menu
2.  Click on ‘Metering’
3.  Click on ‘Meter Input Signal’ (Opt/Alt + i)

**How To Check Audio Hardware Settings**

1.  Click on ‘Audition’ in the top menu
2.  Click on ‘Hardware Settings'
3.  Confirm Default Input is set to microphone or audio interface
4.  Confirm Default Output is set to speakers or audio interface

**  
How to Record:**

1.  Press the Red Record Button or Shift + Spacebar to start recording
2.  Press Stop or Spacebar to Stop Recording
