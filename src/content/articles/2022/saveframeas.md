## After Effects CC 2022: How To Save Frame As (PSD, JPEG, PNG)

**Description:  
**This tutorial will show you how to Save a Frame of the Composition in Adobe After Effects CC 2022 as Photoshop Layers (PSD File) or Jpeg or PNG that can be exported as a file and then re-imported into the project.

**Software Used:  
**Adobe After Effects CC 2022 ([Click To Try or Buy](https://mikemurphy.co/adobe%20(affiliate%20link)))

**How To Save A Composition Frame As A Photoshop PSD File:**

1.  Select the Composition Window or Timeline 
2.  Go to the Composition Menu at the top
3.  Select ‘Save Frame As’
4.  Select ‘Photoshop Layers’
5.  Enter Name of File
6.  Click on ‘Save’

**How To Save A Layer As a PNG Image File:**

1.  Select the Layer to Save Frame As 
2.  Go to the Composition menu at the top
3.  Select ‘Save Frame As’
4.  Select ‘File’
5.  In the Render Queue go to ‘Output Module’ and click on ‘Photoshop’ or the highlighted text
6.  Click on Format and select ‘PNG Sequence’
7.  Click on the ‘Channels’ drop-down menu
8.  Select ‘RGB + Alpha’
9.  Click ‘Ok’
10. Go to ‘Output To” and click on the highlighted text
11. Enter the Name of file
12. Select directory where you want to save the file
13. Click ‘Save’
14. Click on ‘Render’ to export or Save Frame as PNG
15. Navigate to directory to locate the exported PNG file
16. Drag in to the Project Panel to re-import into After Effects
