## After Effects CC 2022: How To Create Shapes From Text

**Description:  
**This tutorial will show you how to convert text layers to shape layers to add animation and effects to individual characters.

**Software Used:  
**Adobe After Effects CC 2022 ([Click To Try or Buy](https://mikemurphy.co/adobe%20(affiliate%20link)))

**Keyboard Shortcut:**

- Cmd/Ctrl + T (Add Text)

**How To Convert Text To Shape Layers:**

1.  Add text to Composition (Cmd/Ctrl + T)
2.  Select text layer in the layers panel
3.  Right-click and select ‘Create’
4.  Then select ‘Create Shapes from Text’
5.  The Text Layer will be hidden (the eyeball is not visible)
6.  The Shape Layer will have a Star icon next to it
7.  The Shape Layer will be renamed with ‘Outlines’ in the title (rename if you want)
8.  Select the Shape Layer Outlines to select the whole group
9.  Twirl open the Shape Layer Outlines to select individual characters

**  
How To Change The Fill Color of a Text Character:**

1.  Twirl open the Shape Layer Outlines to select individual characters
2.  Select the letter 
3.  Twirl open properties
4.  Select Fill and twirl it open 
5.  Double click on the color swatch or select the eyedropper to pick a color
6.  The color of the character should change
