**How To Set A Shape Layer (Circle) as a Track Matte AE 2023:**

1.  Select a layer or multiple layers
2.  Click on Toggle Switches & Modes to see ‘Track Matte’ column in Layers Panel
3.  Click the drop-down menu in the Track Matte column
4.  Select the Shape Layer (or the layer you want to set as a track matte)
5.  The visibility of the Track Matte Layer (shape) will be hidden automatically
6.  All layers set to ‘Track Matte’ will be confined to the bounds/shape of the layer
