## Ecamm Live: How To Enable Live Demo Mode

**Description:  
**

This tutorial will show you how to enable or go into ‘Live Demo Mode’ in the live-streaming software Ecamm Live. When in Live Demo Mode, Ecamm Live will show your entire screen including the Ecamm Live utility windows.

**Why use Live Demo Mode?**

- To share a behind the scenes look of the Ecamm Live production windows with your audience
- To make tutorials about Ecamm Live or to show Ecamm Live utility windows for a livestream tutorial

**Software Used:  
**Ecamm Live ([Click To Try or Buy](https://mikemurphy.co/ecammlive))

**Keyboard Shortcut for Live Demo Mode:  
**Shift + Command + D

**How To Enable Live Demo Mode**

1.  The easy way is to use the keyboard shortcut Shift + Command + D or
2.  Go to the top menu and click on ‘Options’
3.  Click on ‘Live Demo Mode’
4.  Check the box ‘Don’t ask again’
5.  Click on ‘Turn on Live Demo Mode’
6.  You will see a red border around your monitor indicating you are using ‘Live Demo Mode’

**How To Exit Live Demo Mode**

1.  The easy way is to use the keyboard shortcut Shift + Command + D or
2.  Go to the top menu and click on ‘Options’
