## Screenflow 10: How To Change Clip Speed

**Description:  
**This tutorial will show you how to create custom keyboard shortcuts in Screenflow 10.

**Note:  
**To create custom keyboard shortcuts or edit/delete default keyboard shortcuts, you must create a custom keyboard set.

**Keyboard Shortcut:  
**To Open Preferences….Cmd + Comma

**Software Used:  
**Screenflow 10 ([Click To Try or Buy](https://mikemurphy.co/screenflow))

**How To View All Default Keyboard Shortcuts in Screenflow 10:**

1.  Open Screenflow
2.  Go to Screenflow in top Menu
3.  Click on Preferences (or use keyboard shortcut Cmd + Comma)
4.  Click on the 4th icon at the top labeled ‘Shortcuts’

**  
How To Create A Custom Keyboard Shortcut Set:**

1.  Open Screenflow
2.  Go to Screenflow in top Menu
3.  Click on Preferences (or use keyboard shortcut Cmd + Comma)
4.  Click on the 4th icon at the top labeled ‘Shortcuts’
5.  Click on the drop-down menu and select ‘Configure’
6.  Click the + icon
7.  Enter a title for the Set
8.  Click ‘Ok’ to save the set

**  
How to Create Custom Keyboard Shortcuts:**

1.  Open Screenflow
2.  Go to Screenflow in top Menu
3.  Click on Preferences (or use keyboard shortcut Cmd + Comma)
4.  Click on the 4th icon at the top labeled ‘Shortcuts’
5.  Click the drop-down menu and select a custom keyboard set (not the default set)
6.  Search for a keyboard shortcut or browse by category
7.  Select keyboard shortcut to edit
8.  Click in the far right space
9.  Click the X to delete or remove existing
10. Type in a new keyboard shortcut
11. If you see an warning/exclamation, try another keyboard combination
