### What is a Photoshop Artboard?

- Artboards are special layer groups that act as individual canvases inside one Photoshop Document.
- Artboards can be any size and not limited to the size of the main document
- Artboards live on an infinite canvas so you can add as many artboards as you want inside of one Photoshop document
- You **can** add layers and layers groups within artboard groups
- You **cannot** add artboards within another artboard group.
- Artboards are great for mock-ups or design multiple versions or sizes all within one Photoshop .psd file.
- You can export individual artboards or all at once in various formats.

![](https://cdn-images-1.medium.com/max/800/1*31MPTHelq1F1kPm2hdVlHA.png)

Artboards can contain layers & layer groups but not other artboards.

### How To Create Artboards & Your Own Template

Create your own artboard template or multiple templates for sizes that you use over and over again. Save it as a .psd file and re-use it over and over again by duplicating the original or Saving As a new document.

#### Create a New Photoshop Document:

1.  Open Adobe Photoshop
2.  Create New Document (CMD + N)
3.  Enter Dimensions & Resolution for one of the artboard sizes you want in your template
4.  Check the box for Artboards
5.  Click Create

![](https://cdn-images-1.medium.com/max/800/1*VLsRr-EB70vOKwUUQc7jRA.png)

#### How To Rename Artboards:

1.  Double-click the Artboard Group Name in Layers Panel
2.  Artboard on canvas will update when the Layer Panel title changes

![](https://cdn-images-1.medium.com/max/800/1*HHT6RtROrVGtz61GGzfGBQ.png)

Re-Name Artboards

#### How to Ungroup Artboards (Shift + Cmd + G):

If you create artboards but want to make them typical Photoshop layers: Right-click and select ‘Ungroup Artboards’.

#### How To Resize Artboards:

1.  Select Artboard and click and drag the bounding box
2.  Use the Property Panel (Window Menu \> Properties) to enter dimensions and the artboard size will update. **Hint:** Use ‘px’ for pixel and ‘in’ for inches.

![](https://cdn-images-1.medium.com/max/800/1*zQs8kwSUTsNC4WATf5u6cg.png)

Resize Using the Property Panel

#### How To Add Additional Artboards (3 different ways):

1.  Duplicate any Artboard Group in the Layers Panel(Cmd + J)
2.  Select the Artboard Tool under the Move Tool and then click on any Plus (+) symbol
3.  Select the Title of any Artboard and then click on any Plus (+) sign

#### How To Duplicate Artboards:

1.  Hold the Option or Alt key and click and drag on an artboard to duplicate

![](https://cdn-images-1.medium.com/max/1200/1*rq84tX3mly50tSzBursxcg.gif)

To Add A New Artboard: Click Title and Plus (+) Symbol

#### 🔥 Hot Tips:

1.  Press Cmd + Option (Mac) and select a layer to zoom into the Artboard
2.  Press the Spacebar for Hand Tool to move Artboards around on the canvas
3.  Hold Option Key (Mac) or Alt Key (Win) and drag to duplicate an artboard
4.  Use Artboard Presets in the Properties Panel to get ‘inspired’ while building your own artboard template.

#### How to Export Artboards:

1.  Right-click on any Artboard in the Layers Panel and select Export to export individual artboards or layers.
2.  Go to File Menu and then Export…Artboard to Files to export multiple artboards.

![](https://cdn-images-1.medium.com/max/800/1*8i4w2X8Kg3To3qvFZ5PBBw.png)

Export (File \> Export\>Artboards to Files)
