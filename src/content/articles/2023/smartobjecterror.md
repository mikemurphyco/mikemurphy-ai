**How To Fix The Hand-Drawn Effect From Stopping in Adobe After Effects CC 2023:**

1.  Open Photoshop .psd Document
2.  Go To File Menu
3.  Select ‘Package’
4.  Select Destination to save ‘Package’
5.  Click ‘Choose’
6.  Locate 'Package' Folder on Hard Drive
7.  Open After Effects
8.  Double-click in Project Panel
9.  Select Photoshop PSD File to import
10. Import as ‘Composition- Retain Layer Sizes’
11. Check ‘Create Composition’
12. Click ‘Open’
13. Repeat the same Import As settings in the next dialogue box
14. Composition- Retain Layer Sizes’
15. Tick ‘Editable Layer Styles’
16. Click ‘OK’
17. Error Message should not appear
18. Double-click to open Composition
19. Press Spacebar to Preview Animation
20. Save After Effects Project
