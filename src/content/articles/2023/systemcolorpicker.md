## Adobe After Effects CC 2023: How To Use System Color Picker

**Description:  
**This tutorial will show you how to change the default Color Picker to the System (computer) Color Picker instead of using the default color picker in Adobe After Effects CC 2023. One advantage of using the System Color Picker is to save custom colors making it easy to access frequently used colors.

**How To Change To System Color Picker in Adobe After Effects CC 2023:**

1.  Open After Effects
2.  Go to After Effects in the Top Menu
3.  Click on ‘Settings’
4.  Click on ’General’
5.  The the box ‘Use System Color Picker’
6.  Click ‘OK’
7.  Select a layer with Color Fill Effect or Text Layer
8.  Click on the Color Swatch
9.  System Color Picker is now the default
10. Go to ‘After Effects Settings’ to revert back to default AE Color Picker
