## Adobe After Effects CC 2023: Pen Tool Tips for Tracing Text

**Description:  
**This tutorial will show you some tips and tricks for using the Pen Tool to trace letters in Adobe After Effects CC 2023.

**Keyboard Shortcuts:  
**F2: Deselect All  
G: Pen Tool  
Shift + Click: Deselect Path Vertex (this is the trick and key to entire tutorial)

**How To Add Text Layer**

1.  1.  Open New Composition
    2.  Click on T to add Text
    3.  Use Properties Panel to change Font & Size
    4.  Select Text with Selection Tool (V)
    5.  Use Align Panel to center text in Composition

**Add Text Layer**

1.  1.  Open New Composition
    2.  Click on T to add Text
    3.  Use Properties Panel to change Font & Size
    4.  Select Text with Selection Tool (V)
    5.  Use Align Panel to center text in Composition

**How To Use Pen Tool To Trace Over Text by creating Lines:**

1.  1.  Deselect All Layers (F2)
    2.  Select Pen Tool (G) in the top tool bar.
    3.  Click Fill Color and set to None
    4.  Click color swatch for Stroke and pick a color
    5.  Change Stroke Width to desired amount (I used 5px in tutorial)
    6.  Click to add a point
    7.  Click again to add line

**How To Change Directions with Pen Tool (Such as crossing the letter H):**

1.  1.  Draw a line with Pen Tool
    2.  Press the Shift Key
    3.  Click on the selected Vertex/Point/Node (square with color fill)
    4.  Move to new place where you want to draw another line
    5.  Click to add points
