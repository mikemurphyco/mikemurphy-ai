## Adobe After Effects CC 2024: How to Use LoopOut + Wiggle Expression

**Description:  
**This tutorial will show you how to use both the loopOut + Wiggle Expression on the same transform property in Adobe After Effects CC 2024.

**Note:  
**This tutorial uses the loopOut(“pingpong”) and the Position Property, but it will work with other loopOut expressions and transform properties.

**How To Use PingPong Expression (tutorial):  
**<https://youtu.be/aMOBc-e82Cg>

**Expressions Used:  
**loopOut(“pingpong”);  
wiggle (6,10);

**Variables:  
**Container used to define expression

loop = loopOut(“pingpong”);  
wiggle = wiggle (6,10);

**Final Formula  
**loop + wiggle -value

**How To Combine loopOut + Wiggle Expression**

1.  Create variable for loopOut Expression (loop = loopOut(“pingpong”);
2.  Create variable for Wiggle Expression (wiggle = wiggle(6,10);
3.  Create formula to add variables together (loop + wiggle)
4.  Add -Value to reset the position of the layer (-value)
5.  loop + wiggle -value (final formula to combine expressions)
