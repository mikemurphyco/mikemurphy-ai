## Adobe After Effects CC 2023: Increment & Save Version Numbers

**Description:  
**This tutorial will show you how to use Increment & Save in Adobe After Effects CC 2023 to quickly and easily save Project Files with version numbers automatically.

**Keyboard Shortcut:** Increment & Save:  
Cmd + Option + Shift + S (Mac)  
Ctrl + Alt + Shift + S (Windows)

**After Effects Project File Extension:  
**.aep

**How To Use Increment & Save in Adobe After Effects CC 2023:**

1.  Create a new Composition
2.  Add Footage such as image or video
3.  Go to File…Save As and save original in Project Folder
4.  Make changes to AE Project
5.  Go to File…Increment & Save
6.  Repeat by making changes to Project
7.  Select File…Increment & Save and After Effects will save in background and auto-add version numbers

**How To Open Versions**

1.  Go to Project Folder where original AE Project was saved
2.  Double-click on any .aep file to open version in After Effects
