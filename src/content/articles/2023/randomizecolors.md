## Adobe Audition CC 2023: Randomize Multi-Track Colors

**Description:  
**This tutorial will show you how to enable & disable Randomize Track Colors in the Multitrack of Adobe Audition CC 2023.

**Default Track Color Settings:  
**When adding a new track in the Multitrack, each new track will be assigned a random color. If you do not want each track to be a different random color and want all tracks to be the same default Audition Green color, this tutorial will show you how to change the Audition Settings for ‘Randomize Track Color’

**Chapters:  
**00:00  Intro: About This Tutorial  
00:18  Multitrack Defaults: Randomize Track Colors  
00:47  Multitrack To Waveform Colors  
01:20  Multitrack Preferences: Disable Randomize Colors

**Software Used:  
**Adobe Audition CC 2023 ([Click To Try or Buy](https://mikemurphy.co/adobe))

**How To Change Randomize Track Color Default Settings in Adobe Audition CC 2023:**

1.  Open a Multitrack Session
2.  Open Audition Settings (Cmd + Comma) or click on Audition in the top menu
3.  Click on ‘Multitrack Clips’
4.  Go to the section ‘When Inserting Multiple Clips’
5.  Tick the Radio Box ‘Place Each File In Its Own Track’
6.  Click ‘OK’ to commit changes

**  
Note:** Hold the Option/Alt Key while dragging multiple clips to perform the alternative clip insert option.  
**To Clarify**: Whatever is the default setting, holding Option/Alt when dragging multiple clips will do the opposite.
