## Adobe After Effects CC 2023: How To Save Comps As Photoshop Layers

**Description:  
**This tutorial will show you how to export an After Effects Composition as a Photoshop .PSD File that you can modify and add designs in Photoshop and them import the Photoshop File as a New Composition in After Effects CC 2023.

**Software Used:  
**Adobe After Effects CC 2023 ([Click To Try or Buy](https://mikemurphy.co/adobe))

1.  **How To Save Compositions As Photoshop Layers in Adobe After Effects CC 2023:**

    1.  Create or Open a Composition with multiple layers
    2.  Select either the Comp Window or Timeline Panel
    3.  Go to the Composition Menu at the top
    4.  Go to ‘Save Frame As’
    5.  Select ‘Photoshop Layers’ and click
    6.  Change File Name if desired
    7.  Choose destination on hard drive
    8.  Click ’Save’

    **  
    In Photoshop:**

    1.  Open .PSD File
    2.  Add Layers, make changes to design, etc
    3.  Click ‘File’ …Save when done

    **  
    Return to After Effects:**

    1.  Import .PSD File
    2.  Choose ‘Composition…Retain Layer Sizes’
    3.  ‘Editable Layer Styles’
    4.  ‘Create New Composition’ checked
    5.  Double-click to Open New Photoshop-modified comp
    6.  Copy/Paste Layers as needed
