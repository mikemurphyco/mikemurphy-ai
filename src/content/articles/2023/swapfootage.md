## Adobe After Effects CC 2023: How To Swap Out Layer Footage

**Description:  
**This tutorial will show you how to swap out or replace footage from the Project Panel to the Layers Panel while retaining the Layer attributes in Adobe After Effects CC 2023.

**Keyboard Shortcuts:  
**Swap Out Footage (Manually): Option/Alt + Drag  
Swap Out Footage (Automatically) Cmd/Ctrl + Opt/Alt + /  
/ = Slash Key below Question Mark

**How To Swap Out Footage (Manually):**

1.  Create a New Composition
2.  Add layers to create an animation
3.  Select Layer to be replaced with footage from Project Panel
4.  Go to Project Panel
5.  Select Footage
6.  Press Option or Alt
7.  Drag on top of selected layer and let go
8.  Rescale if necessary
9.  New footage should have all of the same property changes and effects.

How To Swap Out Footage (Automatically):

1.  Create a New Composition
2.  Add layers to create an animation
3.  Select Layer to be replaced with footage from Project Panel
4.  Go to Project Panel
5.  Select Footage
6.  Press Cmd or Ctrl + Option/Alt + /
7.  Footage should be replace in Layers Panel
8.  Rescale if necessary
9.  New footage should have all of the same property changes and effects.
