## Adobe After Effects CC 2023: How To Use Color Expression Controls

**Description:  
**This tutorial will show you how to use Color Expression Controls in Adobe After Effects CC 2023 so you can control the color properties of individual or multiple layers all at the same time.

**Keyboard Shortcuts:  
**Cmd/Ctrl + D: Duplicate

**Chapters:  
**00:00  Intro: About This Tutorial  
00:23  Composition Size & Project Overview  
00:43  Creating Shapes  
00:59  Add New Adjustment Layer  
01:12  Add Color Control Expression Control  
01:51  Add Fill Color To Text

02:18  Link Shape & Text to Color Controls  
02:46  Pick-whip Color Properties To Controls  
03:35  How To Animate Color

**Software Used:  
**Adobe After Effects CC 2023 ([Click To Try or Buy](https://mikemurphy.co/adobe))

**How To Use Color Expression Controls in Adobe After Effects:**

1.  Create a New Composition (1920x1080px)
2.  Add Background Layers
3.  Add Shapes with Fill & Stroke colors
4.  Add Text
5.  Add Fill Color Effect on Text Layer
6.  Create a New Adjustment Layer (or Null Object)
7.  Select Adjustment Layer
8.  Go to Effect Menu
9.  Select ‘Expressions’
10. Select ‘Color Control’
11. Go to ‘Effect Control Panel’
12. Select Color Control
13. Tap Enter/Return to rename
14. Duplicate 2 times by right-clicking or pressing Cmd/Ctrl + D
15. Rename Color Controls (Shape_Fill, Shape_Stroke, Text_Fill)
16. Select ‘Color Controls’ layer
17. Lock ‘Effect Controls Panel’ so Color Controls are always visible
18. Select Shape & Text Layers
19. Click in Search field and type ‘Color’ to bring up all color properties
20. Drag the pick-whip next to teach color and drag to the corresponding Color Control in the Effects Control Panel
21. Repeat for all properties
22. Change the colors in the Effect Controls Panel
23. Animate by clicking on the stopwatches
