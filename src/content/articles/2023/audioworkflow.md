## Screenflow 10 To Adobe Audition Audio Workflow

**Description:  
**This tutorial will give you a behind the scenes look at my audio workflow in Screenflow 10 where I export audio only from Screenflow 10 and use Match Loudness in Adobe Audition CC 2023 and then import the audio back into Screenflow 10.

**Summary of Workflow:  
**Edit video in Screenflow 10  
Export audio only from Screenflow 10 as Lossless (.AIFF)  
Open audio file in Adobe Audition CC 2023  
Open ‘Match Loudness’ Panel  
Change Settings to -16LUF  
Run ‘Match Loudness’  
Export audio from Audition as .AIFF file  
Drag audio file from Audition into Screenflow  
Add to Timeline  
Select original audio & mute  
Export video + audio in Screenflow

**Keyboard Shortcut:  
**Cmd + E: Export (Screenflow)  
N: Track Selection Tool (Screenflow)

**Match Loudness Settings:  
**Match To: ITU-R BS.1770-3 Loudness  
Target Loudness: -16 LUFS (Stereo) or -19 LUFS (Mono)  
Max True Peak Level: -1.5 dBTP  
Look-Ahead Time: 12 ms  
Release Time: 200 ms

**Software Used:  
**Screenflow 10 ([Click To Try or Buy](https://mikemurphy.co/screenflow))

**(Screenflow 10) How To Export Audio Only:**

1.  Go to File Menu
2.  Click on Export
3.  Click on Manual
4.  Click on the Preset Drop-down menu
5.  Select ‘Lossless-Audio only’ for an uncompressed AIFF file
6.  Click OK
7.  Name File
8.  Choose Destination
9.  Click ‘Export’

**  
(Adobe Audition) How To Use Match Loudness Panel:**

1.  Open Adobe Audition
2.  Drag audio file exported from Screenflow into the Waveform Editor
3.  Go to Window…Match Loudness to open Match Loudness Panel
4.  Drag audio file from Project Bin on top of the Match Loudness Panel
5.  Click ‘Match Loudness Settings’
6.  Change Match To: ITU-R BS.1770-3 Loudness
7.  Set Target Loudness: -16 LUFS (Stereo) or -19 LUFS (Mono)
8.  Set Max True Peak Level: -1.5 dBTP
9.  Set Look-Ahead Time: 12 ms
10. Set Release Time: 200 ms
11. Click ‘Run’

**  
(Adobe Audition) How To Export Audio from Adobe Audition:**

1.  Go to File….Export
2.  Enter File Name
3.  Select Location
4.  Format: AIFF
5.  Sample Type: Default’\[
6.  Format Settings: Default
7.  **Click ‘OK’**

**  
(Screenflow 10) Open Audio File & Mute Orignal**

1.  Open Screenflow Project
2.  Drag audio file from Audition on to the Timeline
3.  Make sure to align audio clip at the beginning so it syncs with video
4.  Select all original audio clips (Track Select..N)
5.  Go to Inspector Panel
6.  Click on Audio Tab
7.  Check the box for ‘Mute’
8.  Export Screenflow Project
