## Adobe After Effects CC 2024: How To Export PNG Sequences

**Description:  
**This tutorial will show you how to export compositions as PNG or JPEG Image Sequences in Adobe After Effects CC 2024 to generate image files for each frame in an animation.

**Why Export PNG Sequences? **  
This can be useful mixed media animations as you can print image files and hand-draw or paint on each frame and then scan back in to computer and replace frames with modified frames.

**YouTube ShoutOut + Mixed Media Tutorial 👍**:  
[Alan Pun](https://www.youtube.com/@AlanPun)

**Comp Settings Used In Tutorial: **  
1080 x 1920 px (YT Shorts, Insta Stories, etc)  
8 Frames Per Second  
5 Seconds Duration

**How To Compute Exported Frames: **  
Frames Per Second x Duration (ex. 8 fps x 5 seconds = 40 image files)

**How To Export PNG Image Sequences:**

1.  Create New Composition
2.  Create Animation
3.  Go to File Menu
4.  Select ‘Export’
5.  Select ‘Render Queue’
6.  Click link next to ‘Output Module’
7.  Click drop-down next to ‘Format’
8.  Select JPEG or PNG Sequence
9.  If PNG Sequence, go to ‘Channels’
10. Click drop-down and select ‘RGB + Alpha’
11. Go to ‘Output To’ and click on link
12. Choose destination on hard drive to export
13. Click ‘New Folder’ and label
14. Entire title of files
15. Click ‘Save’
16. Locate Folder and files

**Finishing Touches:**

1.  Create New Indesign or Photoshop document US Letter Size (8.5 x11 in)
2.  Place & Resize PNG files (6 per page works well)
3.  Print Pages
4.  Add color, paint, illustrate each frame to taste
5.  Scan pages when design is completed
6.  Add to After Effects
7.  Replace or create new animation with scanned frames
8.  Resize & Position as needed
