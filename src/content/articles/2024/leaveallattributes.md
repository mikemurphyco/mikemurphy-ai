## Adobe After Effects CC 2024: How To PreCompose & Leave All Attributes

**Description:  
**This tutorial will show you how to PreCompose a layer and ‘Leave All Attributes In’ so all effects and attributes will remain in main composition in Adobe After Effects CC 2024.

**Leave All Attributes In: **  
Use this option to create a new intermediate composition. The new composition will become the source to the current layer. 

- Only 1 Layer Can Be Selected for PreComposing
- No Text or Shape Layers

**Keyboard Shortcuts:**  
PreCompose:    
Shift + Cmd/Ctrl + C

**How To Save Frames As Files (Jpegs, Png, PSD, etc)**

1.  Open After Effects Composition
2.  Scrub CTI to Frame
3.  Add Marker (Shift + 8)
4.  Double-click on Marker to add comment
5.  Go to Composition Menu
6.  Select ‘Save Frame As’ File
7.  Go to ‘Render Queue’ panel
8.  Go to ‘Output Module’
9.  Click on drop-down next to ‘Photoshop’
10. Select ‘Custom’
11. Click drop-down menu next to ‘Format’
12. Select ‘JPEG Sequence’
13. Click ‘Format Options’
14. Change JPEG quality to Max
15. Click ‘OK’
16. Resize if desired
17. Click ‘OK’
18. Go to ‘Output To’
19. Click Link to File Name
20. Rename File
21. Select Destination
22. Click ’Save’

**How To Create Render Preset:**

1.  Set ‘Output Module’ Settings as desired
2.  Click on drop-down menu next to ‘Output Module’
3.  Select ‘Make Template’
4.  Enter ‘Settings Name’
5.  Confirm Settings
6.  Click ‘OK’
