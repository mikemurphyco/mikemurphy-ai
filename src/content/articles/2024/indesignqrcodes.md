## Adobe InDesign CC 2024: How To Create QR Codes

**Description:  
**This tutorial will show you how to create a ‘Subscribe To YouTube Channel’ QR Code in Adobe InDesign CC 2024 that people can scan using the Camera App on mobile devices.

**To Use QR Codes:**  
Open Camera App on Mobile Device (iPhone, etc)  
Point at QR Code  
Tap Link on screen 

**YouTube Subscribe Snippet (add after Channel ID) :   
**?sub_confirmation=1

**Example URL:**   
https://youtube.com/mikemurphyco?sub_confirmation=1

**How To Create New Document in Adobe InDesign**

1.  Open Adobe InDesign CC 2024
2.  Click ‘New File’
3.  Change Picas to Inches
4.  Enter 8.5 x 11 inches
5.  Uncheck ‘Facing Pages’
6.  Keep everything else on the default settings
7.  Click ‘Create’

**How To Create YouTube Subscription QR Code in Adobe InDesign:**

1.  Select the Frame Tool (F)
2.  Click on Canvas
3.  Enter 3 in for Width & Height
4.  Click ‘OK’
5.  Click on Selection Tool (V)
6.  Drag selection over Frame
7.  Right-Click
8.  Select ’Generate QR Code’
9.  Select ‘Web Hyperlink’ in Type drop-down menu
10. Enter your YouTube Channel URL (<https://youtube.com/yourchannelID>)
11. Add ?sub_confirmation=1  after Channel ID
12. URL should look like:  
    https://youtube.com/yourchannelID?sub_confirmation=1
13. Click ‘Color’ tab
14. Select a color
15. Click ‘OK’
16. Hover cursor over QR Code to preview the link
17. Right-Click on QR Code and select ‘Edit QR Code’ to edit

**How To Customize & Add Text:**

1.  Click on Selection Tool (V)
2.  Select QR Code
3.  Go to Properties Panel
4.  Change Fill Color to add background or set to ‘None’
5.  Click Color chips to add stroke
6.  Click ’T’ and drag out a Text Box
7.  Enter Type
8.  Use Properties Panel to customize font, size, alignment, etc

**How To Save & Export:**

1.  Click on ‘File’
2.  Click on ‘Save’ (Cmd/Ctrl + S) to Save .indd Document File
3.  Click ‘File’ and the ‘Export’
4.  Enter Title
5.  Select Destination to export file
6.  Choose ‘Format’ type from drop-down (.pdf, jpeg, png, etc)
7.  Click ‘Save’
8.  Choose Range or ‘All’
9.  Change Quality to ‘Maximum’
10. Resolution to 72, 150, or 300
11. Color Space to RGB
12. Check Box for ‘Transparent Background’ if exporting as transparent png
13. Click ‘Export’
14. Share Files
