## Adobe After Effects CC 2025: Shape Layer Types in After Effects

**Description:  
**This tutorial is everything you need to know about the 3 different Shape Layer Types in Adobe After Effects CC 2025. 

**Shape Layer Types:**  
Parametric Shapes (contain adjustable Path parameters)  
Bezier Shape Paths (Path only)   
RotoBezier Shape Paths (Similar to Bezier with automatic curves)

**How To Create Parametric Shapes:**

1.  Select one of the Shape Tools (Rectangle. Rounded Rectangle, Ellipse…)
2.  Select Fill & Stroke Color
3.  Uncheck ‘Bezier’ Checkbox
4.  Drag out a shape
5.  Hold Shift for ‘Perfect’ Square, Circle, etc
6.  Twirl open Shape Layer
7.  Twirl open Path Group to adjust Parameters (Size, Position, Roundness, Radius, etc)

**How To Create Bezier Shapes:**

1.  Select one of the Shape Tools or Pen Tool
2.  Select Fill & Stroke Color
3.  Check ‘Bezier’ Checkbox
4.  Click to add points or Drag out a shape
5.  Hold Shift for ‘Perfect’ Square, Circle, etc
6.  Twirl open Shape Layer
7.  Select Path Group
8.  Use Selection Tool (V) to Shift + Click on Points to customize points

**How To Create RotoBezier Shapes:**

1.  Select Pen Tool
2.  Select Fill & Stroke Color
3.  Check ‘RotoBezier’ Checkbox
4.  Click to add points or Drag out a shape
5.  Twirl open Shape Layer
6.  Select Path Group
7.  Use Selection Tool (V) to Shift + Click on Points to customize points

**How To Convert Parametric to Bezier Shapes:**

1.  Twirl open Parametric Shape Layer
2.  Select Path Group
3.  Right-Click on Path Group
4.  Select ‘Convert to Bezier Path’

**How To Convert Bezier to RotoBezier Shapes:**

1.  Twirl open Bezier Shape Layer
2.  Select Path Group
3.  Right-Click on Path Group
4.  Select ‘RotoBezier’ so there is a checkmark
