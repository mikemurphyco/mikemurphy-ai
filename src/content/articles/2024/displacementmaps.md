**How To Add Displacement Map Effect:**

1.  Add Footage Layer
2.  Add Text Layer
3.  Add or Create Layer To Be Control Layer or Displacement Map
4.  Select Text Layer Layer
5.  Go to Effect Menu…Distort
6.  Select ‘Displacement Map’
7.  Go to ‘Effect Controls’ Panel
8.  Click the drop-down next to ‘Displacement Map Layer’
9.  Select ‘Control’ Layer (Displacement Map Layer)
10. Click ‘Source’ drop-down and change to ‘Effects & Masks’
11. Change Channel (optional)
12. Change values for Horizontal & Vertical Displacement to achieve desired effects

• 13. Finishing Touches: Consider adding ‘Gausian or Fast Box Blur’, ‘Curves/Levels’ and or ‘Tint’ Effect
