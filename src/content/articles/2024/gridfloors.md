**How to Create Floor Using Grid Effect:**

1.  Create New Composition
2.  Add New Solid Layer
3.  Go to ‘Effects & Presets’ Panel
4.  Search For ‘Grid’ in the Generate category
5.  Drag ‘Grid’ effect onto Solid Layer
6.  Go to Effect Controls Panel
7.  Change ‘Size From’ to Width Slider
8.  Change Width to 48
9.  Change Border to 20
10. Change Color to Brown Hex Color (221602)
11. Go to Solid Layer
12. Click the 3D Column to convert to 3D Layer
13. Go to X Rotation
14. Enter value of -82
15. Change Scale to 125
16. Drag Solid to the bottom of Comp Window
17. Customize to taste
18. Add Elements to Scene
19. Have Fun!!
