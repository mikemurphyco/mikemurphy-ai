**How To Access Contextual Type Tool Menu:**

1.  Add Text To Composition
2.  Select Text Layer
3.  Select Type Tool (Cmd/Ctrl + T)
4.  Right-click on Text for Contextual Type Menu
5.  Repeat by selecting Type Tool & Right-clicking on Text

**How To Switch From Text to Selection Tool (Temporarily):**

1.  Add Text
2.  Press & Hold Cmd/Ctrl Key
3.  Click on Text
4.  Let go of Cmd/Ctrl to continue editing or adding Text

**How To Switch From Text to Selection Tool (Method 1):**

1.  Add Text
2.  Tap the Escape Key to switch from Edit Text Mode to Bounding Box
3.  Tap ‘V’ for Selection Tool

**How To Switch From Text to Selection Tool (Method 2):**

1.  Add Text
2.  Tap the Cmd/Ctrl + Return/Enter
