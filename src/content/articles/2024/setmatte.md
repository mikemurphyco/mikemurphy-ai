**How To Use Set Matte Effect in Adobe After Effects CC 2024:**

1.  Create New Composition
2.  Add Layers such as text, shapes, graphics, textures, animated backgrounds
3.  Select a Text Layer or similar
4.  Go to Effects Menu at the top
5.  Go to ’Channel’
6.  Select ‘Set Matte’ to add to the selected layer
7.  Go to ‘Effect Controls’ Panel
8.  Click ont the drop-down menu next to ‘Take Matte from Layer’
9.  Select Layer to use as Matte Layer
10. Click on drop-down and change to ‘Effects & Masks’ if using additional Effects or Masks on layers.
11. Go to ‘Use For Matte’ and select channel such as ‘Alpha Channel or Luminance’
12. Toggle ‘Invert Matte’ and other properties to get the look you want
13. Toggle off the visibility of Matte Layer
14. Add additional layers
15. Copy/Paste Set Matte Effect
