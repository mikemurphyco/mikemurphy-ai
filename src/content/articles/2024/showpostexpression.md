EE: Show Expressions

**Expressions Used: **  
loopOut(“offset”)

**How To Use Show Post-Expression Graph:**

1.  Create New Composition
2.  Create Animation (ex. on Position Property)
3.  Add Expression such as loopOut(“offset”)
4.  Select Position Property to highlight keyframes
5.  Click the icon next to the Equal Symbol to enable ‘Show Post-Expression Graph’
6.  Click on the Graph Editor icon to show Graph Editor (Shift + F3)
7.  Change Graph Editor to ‘Edit Value Graph’
8.  Tick ‘Expression Editor’
9.  Change Expression to view changes in real time.
