## Adobe After Effects CC 2025: How To Create Mixed Media Style Animation

**Description:  
**This tutorial will show you how to create a hand-drawn Mixed Media Style Animation in Adobe After Effects CC 2025 to speed up workflow when working with Text.  

**  
Keyboard Shortcuts:**

Move CTI 10 Frames: Shift + Option + Right Arrow  
Duplicate Layers: Cmd/Ctrl + D

**Font Used in Tutorial:**  
https://fonts.google.com/?query=londrina

**Expression Used:**  
posterizeTime(6):

**How To Prepare Animated Grid Paper Background:**

1.  Create New Composition
2.  Add Grid Paper Texture
3.  Increase Scale of Texture to be bigger than background
4.  Drag CTI to the beginning of the Timeline
5.  Tap P for Position Property
6.  Click Stopwatch to add a keyframe
7.  Right-click on Keyframe and select ‘Toggle Hold Keyframe’
8.  Move CTI forward in time by 10 Frames (Shift + Option + Right arrow)
9.  Manually move texture on Comp Window to a new location
10. Repeat moving CTI 10 Frames and changing Position all the way down the Timeline
11. Tap Spacebar to Preview Animation
12. Add Elements to Scene
13. Change Blend Mode to Color Burn  (or preferred look)
14. Add ‘Hand-Drawn’ Effect to subtle animation of elements

**How To Animate Text Layers:**

1.  Add Text (Londrina Solid Font used)
2.  Twirl open Text Layer
3.  Click on ‘Animate’
4.  Select ‘Position’
5.  Click on ‘Add’
6.  Click on ’Selector’
7.  Click on ‘Wiggly’
8.  Change Y value of Position to 5
9.  Add Layer Style Drop Shadow
10. Tap Spacebar to Preview

**How To Make Cardboard Texture Letters:**

1.  Add Cardboard (or any) texture layer
2.  Add Text Layer (to serve only as a guide)
3.  Select Cardboard layer
4.  Select Pen Tool
5.  Click Points and trace around text layers
6.  Repeat for all letters
7.  Delete Text Guide Layer to reveal cardboard letters in hand-drawn style

**How To Animate Hand-Painted Texture (Offset Effect):**

1.  Add Hand-Painted Texture
2.  Go to ‘Effects & Presets’ Panel
3.  Search for Offset Effect and drag on to Texture Layer
4.  Option + Click on stop-watch next to ‘Shift Center To’
5.  Enter Expression:  
    posterizeTime(6):  
    wiggle(5,5000)
6.  Press SpaceBar to preview

**How To Use Track Mattes:**

1.  Go to Animated Texture Layer
2.  Go to ‘Track Matte Column’
3.  Click the drop-down and select ‘Text’ Layer

**How To Create ‘Sticker’ Page Turn Effect:**

1.  Select Pre-Comp of Text Layer
2.  Go to ‘Effects & Presets’ panel
3.  Search for ‘CC Page Turn’ effect
4.  Drag CTI forward
5.  Click on stop-watch next to ‘Fold Position’ to add a keyframe (end of animation)
6.  Drag CTI to the beginning of the Timeline
7.  Change Fold Position to starting position (fold to the left)
8.  Tap SpaceBar to preview
