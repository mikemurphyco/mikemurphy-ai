**How to Combine Multiple After Effects Projects:**

1.  Create New Folder
2.  Copy & Paste Footage Folder from Master Project Folder
3.  Open After Effects to create ‘New Untitled Project’
4.  Go to File…Save As…Save As
5.  Enter Name of Project (i.e. All Lessons)
6.  Go to File…Import or double-click in an empty space in Project Panel
7.  Select all .aep Project Files
8.  Click ‘Open’ to Import
9.  Twirl open Folders
10. Double-click on Compositions to Open
11. Go to File Menu…Save
12. Go to File Menu…Close Project
13. Double-click on Project (.aep) to re-open at any time.
