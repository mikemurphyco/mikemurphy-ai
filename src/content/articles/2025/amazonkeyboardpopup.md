**About This Tutorial:  
**This tutorial will show you how to stop the annoying Keyboard Shortcut Popup when visiting Amazon in a web browser on your Mac.

**Problem:  
**When you visit Amazon, a small popup related to Keyboard Shortcuts automatically appears.

**The Fix (MacOS):  
**Toggle Off ‘Full Keyboard Access’ in Accessibility Settings…Keyboard

✅ **What Causes the Popup?  
**This issue happens when a setting called Full Keyboard Access is turned on in your Mac's Accessibility Settings.  
This feature allows you to navigate your computer using just the keyboard, but it can trigger popups like the one you see on Amazon.

🛠️ **How to Turn Off Full Keyboard Access on a Mac:**

1.  Click the Apple logo in the top-left corner
2.  Select System Settings
3.  Select Accessibility in the left sidebar
4.  Select Keyboard on the right side
5.  Locate ‘Full Keyboard Access’ in the top right
6.  Toggle Off the Full Keyboard Access Switch
7.  To test, open a web browser and visit [Amazon](https://amazon.com)
8.  If Full Keyboard Access is Off, Popups on Amazon should no longer appear

[![Mac Settings](/assets/media/2025/01/01_MAC-SYSTEM-SETTINGS-985x1024.png)](/assets/media/2025/01/01_MAC-SYSTEM-SETTINGS.png) Open MacOS System Settings...Accessibility...Keyboard

 

 

 

[![Cause of Popups](/assets/media/2025/01/02_Cause-of-Popups-928x1024.png)](/assets/media/2025/01/02_Cause-of-Popups.png) Locate 'Full Keyboard Access'. If Toggled On, Amazon Popups will automatically appear.

 

 

 

 

[![Fix: Full Keyboard Access](/assets/media/2025/01/04_The-Fix-950x1024.png)](/assets/media/2025/01/04_The-Fix.png) Toggle off 'Full Keyboard Access' to disable Amazon Keyboard Shortcut

 

 

 

 

🎯 **What If You Want to Keep Full Keyboard Access On?  
**If you have Accessibility needs and want Full Keyboard Access On your Mac, but want to toggle off Amazon popups:  
When visiting Amazon using a web browser: use the keyboard shortcut ***Shift + Option + Z** to toggle on/off.*

✅ **Problem Solved!  
**Turning Full Keyboard Access off will stop the Amazon keyboard shortcut popup from appearing.  
This fix works on all versions of macOS, including the latest macOS Sequoia.
