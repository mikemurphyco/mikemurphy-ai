**How to Install Claude Code:**

1.  Upgrade Claude.ai account to Pro or Max Plan
2.  Click Apple in top left corner of Mac
3.  Confirm MacOS is 10.15 or later
4.  Open Terminal (located in Utilities Folder in Applications Folder)
5.  Confirm Node.js is installed by typing node —version in Terminal
6.  You will see version number if installed
7.  If Node.js is not installed, visit <https://nodejs.org> to download and install
8.  Re**-**confirm Node.js is installed by typing node —version in Terminal
9.  Clear Terminal (type ‘clear’ or Cmd + K)
10. Install Claude Code by pasting snippet of code:  
    npm install -g @anthropic-ai/claude-code
11. Confirm installation by typing claude —version
12. Confirm health of install by typing claude doctor
13. Authenticate Preferred Login by selecting 1 for Web (most common)
14. Select 2 for using Claude Code API only
15. Tap Return to authenticate
16. Select Yes to setup Claude Code Terminal Optimization
17. Enter /help to view Claude Code commands
18. Tap /exit to exit the Repl

**How to Run Claude Code (Demo):**

1.  Create a Project Folder in Home Directory (or Desktop, etc)
2.  Open Terminal
3.  Use Change Directory (cd) to navigate to project folder
4.  Tip: Right-click on Folder and go to Services….New Terminal at Folder
5.  Type claude to open Claude Code
6.  Type anything you want Claude Code to do
7.  Create a cool, retro/vintage interactive data visualization with the phrases ‘Mike Murphy’ and ‘AI Handyman’ floating around
8.  Chat with Claude Code to make changes
9.  /exit to exit Claude Code
