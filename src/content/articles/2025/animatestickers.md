🌹 **Generate PNG Stickers**

1.  Open [ChatGpt.com](http://ChatGpt.com)
2.  Enter Prompt with text, color & transparent background in ‘Sticker Generator Custom GPT’
3.  Hover cursor over graphic and click ‘download’ arrow in the top-right corner

🌹 **Animate PNG Stickers**

1.  Create New Composition (1080x1080, 5 seconds)
2.  Import PNG Sticker
3.  Drag PNG Sticker on Comp
4.  Press P for Position
5.  Option + Click and add ‘looping wiggle expression’
6.  Adjust frequenct, amp, time (set to comp duration)
7.  Press R for Rotation
8.  Option + Click and add ‘looping wiggle expression’
9.  Adjust frequency, amp, time (set to comp duration)

🌹 **Export PNG Stickers**

1.  Go to File Menu
2.  Export
3.  Select ‘Add to Adobe Media Encoder Queue’
4.  Duplicate Output
5.  Search ‘Preset Browser’ for Animated GIF w/transparency and drag 1st output
6.  Search for ‘4444’ and drag Apple ProRes 4444 w/transparency on 2nd output
7.  Change Output File Location (where to save)
8.  Click ‘green play icon’ to Start Queue
9.  Done!
