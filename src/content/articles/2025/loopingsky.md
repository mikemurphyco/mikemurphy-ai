## Adobe After Effects CC 2025: Looping Sky & Clouds Background Animation

**Description:  
**This tutorial will show you how to create a seamlessly looping sky & clouds background animation using the Offset Effect in Adobe After Effects CC 2025.

**How To Create Sky & Clouds:**

1.  Create New Comp
2.  Add Scanned Textures or Create New Solid Layers
3.  Select Layer to be used for clouds
4.  Select Pen Tool and create the shape of clouds
5.  Set Mask Mode to Add
6.  Go to Effects & Preset Panel
7.  Search for Curves and add to Sky Layer to boost color & contrast
8.  Search for ‘Hand-Drawn Effect’ and add to cloud layer for rough edges & wiggle

**How To Create Sky & Clouds Animation:**

1.  Create New Comp
2.  Drag in Sky Comp 1 into new Comp
3.  Go to Effects & Preset Panel and search for ‘Offset’
4.  Drag Offset onto Sky Comp Layer
5.  Drag CTI to the beginning of the Comp
6.  Click the Stopwatch next to ‘Shift Center To’ property to add a keyframe
7.  Move CTI to 12 second mark
8.  Click in the X value of Offset Effect
9.  Enter -1920 after the original value to set second keyframe at -960
10. Tap Spacebar to Preview Animation
11. Add additional graphics elements to flush out the scene (mountains, trees, etc)

**How To Add LoopOut Expression:**

1.  Press Option/Alt and click on the stopwatch next to ‘Shift Center To’ property
2.  Type the Expression loopOut()
3.  Tap Spacebar and animation should loop endlessly

**Finishing Touches:**

1.  Create New Composition (Main)
2.  Drag Sky 2 Comp into the Comp
3.  Add other elements to the scene (graphics, animations, etc)
4.  Tap Spacebar to Preview
5.  Export Video & Enjoy!
