## How To Update Hostinger VPS Packages (Ubuntu OS)

**Description:**  
In this tutorial, you will learn how to update the Ubuntu OS Packages on your Hostinger VPS to keep the OS clean, secure and running efficiently.

**In this tutorial, you'll learn:**

- What are Ubuntu OS Packages?
- How to check for available package updates?
- How to install package updates?
- How to clean up and auto-remove unused packages
- How to confirm that everything is updated on the Ubuntu OS?

**Hostinger VPS:**  
[https://mikemurphy.co/vps](https://mikemurphy.co/vps)

——————

1.  To see which package updates are available, run: `apt list --upgradable`
2.  Refresh the list of available updates...“checking for new updates. `sudo apt update`
3.  Install the updates `sudo apt upgrade`
4.  Clean up unused packages `sudo apt autoremove`
5.  Confirm everything is updated `apt list --upgradable` `listing...done`
