## How To Update Diffusion Bee Using GitHub

**Description:  
**In this tutorial, you will learn how to update DiffusionBee on GitHub so you can use Flux Models locally on your Mac. Diffusion Bee is MacOS only and is the easiest way to run AI Models with Stable Diffusion run locally on your Mac. It is private, no internet required and is 100% free.

**Problem This Tutorial Solves:**   
If you download DiffusionBee on www.diffusionbee.com, the downloaded version is 2.5.1 and you will see warnings on the Models page of the app that you must update the application to run Flux Models. The problem is you cannot update DiffusionBee from the app, you must go to GitHub and replace the 2.5.1 with 2.5.3.

**Download Diffusion Bee (on GitHub):**   
https://github.com/divamgupta/diffusionbee-stable-diffusion-ui/releases/tag/2.5.3

🪜**How To Update Diffusion Bee using GitHub**

1.  Quit Diffusion Bee
2.  Go to [DiffusionBee.com](http://DiffusionBee.com)
3.  Go the footer of the website
4.  Click on ‘GitHub’ (see link above)
5.  On GitHub to to the right sidebar
6.  Look for Latest Releases
7.  Click on 2.5.3
8.  Select Mac Chip Type:  MacOS Silicon or Inel
9.  Double-click to install
10. Drag DiffusionBee into Applications Window
11. Yes to Replace
12. Re-open Diffusion Bee
13. Click on DiffusionBee in top menu

• 14. Click ‘About’ and confirm version is 2.5.3**
