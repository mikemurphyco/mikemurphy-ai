## How To Open VS Code From Terminal with Code Dot

**Description:**  
In this tutorial, you will learn how to open any directory from the Terminal in Visual Studio Code (VS Code) using Code + Period.

**Download VS Code:**  
[https://code.visualstudio.com/](https://code.visualstudio.com/)

**Keyboard Shortcuts:**  
Code . (Code + Period)  
Show All Commands (Shift + Cmd + P)

**Show All Commands:**  
Install 'Code' command in PATH

**In this tutorial, you'll learn:**

1.  What is Code + Dot?
2.  How To Show All Commands (Shift + Cmd + P)
3.  How to install Code Command in VS Code
4.  How to uninstall Code Command in VS Code
5.  How to Open Directories from Terminal in VS Code
6.  Helpful Terminal Commands

🪜 **How To Open VS Code Using Code + Dot in Terminal:**

1.  Confirm Code Dot is Enabled
2.  Open Project Directory in Terminal
3.  Enter Code + . (if VS Code opens, then it is enabled)
4.  Show All Commands (Shift + Cmd + P)
5.  Click 'Install 'Code' command in PATH' to install Code + Dot
6.  Open directory in Terminal
7.  Press Code + Period and VS Code will open
