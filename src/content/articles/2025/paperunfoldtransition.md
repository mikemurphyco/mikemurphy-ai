**How To Set Up Stop Motion:**

1.  Open Stop Motion Studio Pro App on iPhone or iPad
2.  Use Blue Tack adhesive to secure background and one side of paper
3.  Rip paper. Take Photo. Repeat
4.  Export stop motion capture as images

**Process & Edit Photos:**

1.  Import all photos into Lightroom or Photo Editor of choice
2.  Convert to Black & White
3.  Export all edited images

**Import Images to Photoshop:**

1.  Create new Photoshop document (1920 x 1080)
2.  Go to File...Scripts…Load Files Into Stack
3.  Select all edited images to import as layers

**Remove Background in Photoshop:**

1.  Select Layer
2.  Select Object Selection Tool (W)
3.  Hover cursor over paper until pink stroke appears around edges
4.  Click to make a selection
5.  Press Cmd/Ctrl + J to jump to new layer
6.  Delete layer below
7.  Repeat for all Layers

**Export PNG Images from Photoshop:**

1.  Go to File…Export
2.  Select Layers To Files
3.  Select Destination to Save
4.  Add Prefix Title
5.  Select ‘PNG 24’ for Type
6.  Check ‘Transparency’
7.  Uncheck ‘Trim Layers’
8.  Click ‘Run’

**Import PNG Sequence into After Effects:**

1.  Open After Effects
2.  Go to File…Import…File
3.  Select first image…Press Shift and select last image
4.  Check ‘Create Composition’
5.  Uncheck ‘Multiple Compositions’
6.  Click ‘Open’
7.  Create ‘Single Composition’
8.  Still Duration: 0:00:00:02 (2 frames)
9.  Check ‘Sequence Layers’

**How To Create Paper Tear Transition:**

1.  Create New Comp called ‘Main’ (1920 x 1080, 24fps, 10 secs)
2.  Add Paper PNG Sequence
3.  Right-click..Time…Enable Time Remapping
4.  Go to 2nd keyframe
5.  Press Cmd/Ctrl + Left arrow to go back 1 frame
6.  Click diamond to add a keyframe
7.  Delete the last keyframe
8.  Extend layer to fill timeline
9.  Option/Alt + Click Time Remap Stopwatch
10. Enter loopOut() Expression to loop animation
11. Create New Comp called ‘Placeholder’ and add footage
12. Return to ’Main’
13. Select Placeholder Layer
14. Go to Track Matte Column (Right-click Header…Modes)
15. Select ‘Paper Tear’ Comp as Alpha Matte
16. Add ‘Hand-drawn’ Effect + Drop-Shadow
17. Tap Spacebar to Preview
