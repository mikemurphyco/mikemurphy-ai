**Prepare For Print:**

1.  Open After Effects
2.  Import Video Footage
3.  Create New Comp From Footage
4.  Set Work Area To Be Exported…Beginning (B) & End(N)
5.  Add To Render Queue (Composition Menu)
6.  Click ‘Best Settings’ change to ‘Work Area’
7.  Click  ‘Output Module’ change to PNG, JPEG or Photoshop Sequence
8.  Click ‘Output To’ to set export destination

**Print:**

1.  Open Adobe Bridge
2.  Go to ‘Output’ Tab
3.  Change Output Settings Template to 2x2 Cells
4.  Click ‘Export To PDF’
5.  Export PDF
6.  Print All Frames (Cmd/Ctrl + P)

**Mixed Media & Scan**

1.  Number each page to keep in order
2.  Mark up each page (draw, paint, collage, etc)
3.  Have Fun!!!
4.  Scan Frame Sheets into computer
5.  300 dpi, JPEG

**After Effects:**

1.  Import Edited Frames into After Effects
2.  Create a new Composition the same size as original footage
3.  Duplicate layer 3x for a total of 4 Layers Per Scan
4.  Trim Layers to 1 Frame Each
5.  Offset Each Layer
6.  Drag CTI to each layer on Timeline and adjust image to fit
7.  Repeat for all pages
8.  Add Effects or Adjustment Layers (i.e. Curves)
9.  Close Comp
10. Go to Main Comp with original footage
11. Drag PreComp into Main Comp
12. Duplicate to extend duration
13. Add Music
14. Tap Spacebar
15. Enjoy
