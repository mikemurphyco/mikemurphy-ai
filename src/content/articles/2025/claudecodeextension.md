🪜 **How To Import N8N-MCP Server into Claude Code:**

1.  Open VS Code
2.  Click Extensions icon in left sidebar
3.  Search For Claude Code
4.  Click 'Install'
5.  Look for Claude Logo in top right to launch Claude Code
6.  Open Project Folder
7.  Chat With Claude Code
8.  Open Extension Settings to switch to Terminal
9.  Have fun!
