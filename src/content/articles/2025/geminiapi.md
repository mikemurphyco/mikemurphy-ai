🪜 **How to Generate Google Gemini API Keys:**

1.  Open a web browser and visit [aistudio.google.com](http://aistudio.google.com)
2.  Sign-in using Google account
3.  Click ‘Dashboard’ in the left sidebar
4.  Click on ‘API Keys’
5.  Click on ‘Create API Key’ in the top right
6.  Copy API Key
7.  Save in 1Password or similar
8.  Click ‘Set Up Billing’
9.  Follow prompts to set up billing
10. Copy API Key to clipboard
11. Visit [Hostinger.com](http://Hostinger.com) and sign into VPS
12. Click ‘Manage’ & ‘Manage App’
13. Sign in to n8n
14. Click Plus (+) icon
15. Click ‘Credentials’
16. Paste Google Gemini API Key
17. Click ‘Save’
18. X out to close Dialogue Box
19. Click Plus(+) to add new node
20. Search for Gemini
21. Add Gemini Chat Node
22. Click dropdown to select Gemini API Key
