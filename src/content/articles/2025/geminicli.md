## How to Install Google Gemini CLI in MacOS Terminal

**Description:**  
In this tutorial, you will learn how to create Google Gemini CLI on MacOS to run Gemini 2.5 Pro Model in the Terminal.

**In this tutorial, you'll learn:**

- What is Gemini CLI?
- How to install Node.js & Homebrew
- How To Install Gemini CLI
- How to Authenticate Gemini CLI
- How To Launch & Quit Gemini CLI
- How To Access /help & /search

**What is Gemini CLI?**  
Gemini CLI is a free, open-source AI agent developed by Google that brings the capabilities of the Gemini model directly into the command-line terminal. It provides lightweight and powerful access to Gemini for coding, automation, research, and content generation, allowing users to interact with Gemini entirely from their local development environment.

**Install via HomeBrew:**  
`brew install gemini-cli`

**Is HomeBrew Installed?**  
`brew -v`

**Is Node.js Installed?**  
`node -v`

🪜 **How to Install Gemini CLI on MacOS:**

1.  Open MacOS Terminal
2.  Confirm HomeBrew is Installed (brew -v)
3.  Confirm Node.js is Installed (node -v)
4.  Install Homebrew and/or Node.js if not installed
5.  Install Gemini CLI via Homebrew (`brew install gemini-cli`)
6.  Open Terminal
7.  Enter 'gemini' to launch Gemini CLI
8.  Select Authentication Method (Login with Google)
9.  Tap Return
10. Select Google Account and 'Sign In'
11. Return To Terminal to confirm Gemini CLI is working
12. Enter a test message (How Do I Use /search tool?)
13. Press 'clear' to clear window
14. Press /help for Help
15. Press /search to Search Inside Files, File Path, Web
16. Press /quit to quit and exit Gemini CLI
