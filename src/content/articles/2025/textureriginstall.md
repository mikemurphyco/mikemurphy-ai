# The Animation Texture Rig:

### *How To Install & Set Up*

------------------------------------------------------------------------

In this video, I show you how to install and use the Animated Texture Rig from Mike Murphy AI Build 107 where I used Cursor AI + Chat GPT to build a digital product from start to finish.

**The steps:**

1.  Copy & Paste Create_Controller.jsx in the Scripts Folder of After Effects (Applications/Programs)
2.  Copy & Paste TextureRig.ffx in User Presets Folder (Documents)
3.  Open AE
4.  Create New Comp
5.  File Menu...Scripts...Select Create_Controller.jsx (This will add Null Object + 5 Slider Controls)
6.  Add Texture or any layer
7.  Select Layer
8.  Go to Effects & Presets
9.  Twirl Open Animation Presets Folder
10. Twirl Open User Presets
11. Double-Click on TextureRig 
12. Tap Spacebar to Preview
13. Select Controller
14. Go To Effect Controls Panel to customize animation

 

**🔹 STEP 1: Install the Controller Script (.jsx)**  
1. Close After Effects  
2. Double-click to open .zip folder  
3. Open After Effects Files (Preset + Controller) Folder  
4. Copy Create_Controller.jsx File (Cmd/Ctrl + C)  
5. Open Applications/Adobe/After Effects \[version\]/Scripts/  
6. Paste (Cmd/Ctrl + V)

**🔹 STEP 2: Install the Preset (.ffx)**  
1. Copy TextureRig.ffx (Cmd/Ctrl + C)  
2. Open Documents Folder…Adobe…After Effects…User Presets  
3. Paste (Cmd/Ctrl + V)

**🔹 STEP 3: Open After Effects**  
1. Create New Composition  
2. Go to File…Scripts…Create_Controller.jsx  
3. This adds Null Object + 5 Slider Controls  
4. Import Textures  
5. Add to Composition  
6. Go to Effects & Presets Panel  
7. Twirl open Animation Presets…User Presets  
8. Drag TextureRig.ffx on texture or double-click to apply to selected layers  
9. Tap Spacebar to preview!  
10. Select ‘Controller’ Layer  
11. Go To ‘Effect Controls’ Panel to change animation

✅ **Default Recommended Settings:**

\* Position Jitter: 50  
\* Rotation Jitter: 2  
\* Scale Jitter: 1  
\* FPS: 6  
\* Base Scale: 110
