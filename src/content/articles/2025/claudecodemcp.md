🪜 **How To Import N8N-MCP Server into Claude Code:**

1.  Go To GitHub Repo
2.  Copy npx n8n-mcp (snippet)
3.  Open Terminal
4.  Paste in snippet & press Return
5.  Run Claude Code
6.  /init to Create Claude.md File
7.  Return To GitHub Repo & Copy Full Configuration File
8.  Open Claude.md File and Paste Instructions
9.  Check MCP Status (claude mcp list)
10. Enter Prompt in Claude to build n8n workflow and press 'Submit'
11. Copy JSON File when Claude is finished
12. Open N8N and click '+' Workflow
13. Paste JSON Code or click Ellipsis and 'Import File'
14. Set Credentials
15. Vibe Code with Claude Code to fix issues or make changes
16. Test n8n Workflow and activate when finished
17. Rinse & Repeat!
