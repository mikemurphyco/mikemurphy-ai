🛠️ **How To Modify Code & Add Fractal Noise Button Using Cursor AI**

- Open Cursor
- Open Project Folder
- Press Cmd/Ctrl + i to jump cursor to Chat Panel
- Type in Text Prompt asking Cursor to add a button with Fractal Noise effect
- Drag & Drop screenshot of layout mockup
- Press Send
- Accept File generated by Cursor

🛠️ **How To Add Code (.jsx) to After Effects Script UI Panel**

- Open Project Folder
- Open ‘dev’ folder
- Copy .jsx file (Cmd/Ctrl + D)
- Click on Applications
- Click on Adobe After Effects \[latest version\]
- Click on ‘Scripts’
- Click on ‘ScriptsUI Panel’ and drag to Sidebar for shortcut
- Paste copied .jsx file in ScriptsUI Panel
- Press ‘Replace’
- Enter Mac Password

🛠️ **How To Test Fractal Noise & Debug in After Effects**

- Open After Effects
- Go to Window Menu
- Select ‘myaetools.jsx’ to open
- Create New Adjustment Layer
- Click the ‘Fractal Noise’ button
- You should see Fractal Noise Effect!

🛠️ **How To Add Additional 11 Effect Buttons**

- Open Cursor
- Open Project Folder
- Press Cmd/Ctrl + i to jump cursor to Chat Panel
- Type in Text Prompt asking Cursor to add 11 additional effects
- Drag & Drop screenshot of layout mockup
- Press Send
- Accept File generated by Cursor
- Replace JSX File
- Open After Effects
- Test Panel

🛠️ **How To Debug & Fix Errors**

- Take a screenshot of error
- Return To Cursor
- Ask Cursor to fix error
- Accept File
- Replace JSX file in After Effects
- Open After Effects
- Test Panel in all of its glory
- Celebrate & smile and enjoy your panel!
