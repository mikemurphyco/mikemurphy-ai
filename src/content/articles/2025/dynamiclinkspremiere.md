**How To Create New Adobe Dynamic Link (Technique 01)**

1.  Open Adobe Premiere Pro
2.  Create a new sequence
3.  Add Footage to Timeline
4.  Go to the File Menu
5.  Select ‘Adobe Dynamic Link’
6.  Select ‘New After Effects Composition’
7.  Enter Name of After Effects (.aep) Project File
8.  Rename Comp in AE (optional)
9.  Add Text Layer
10. Drag CTI to Footage
11. Add Typewriter Effect
12. Return To Premiere Pro
13. Locate new AE Comp in Project Bin
14. Rename (optional)
15. Drag AE Comp on Timeline
16. Return to AE to make changes

**How To Replace Footage with After Effects Composition (Technique 02)**

1.  Open Adobe Premiere Pro
2.  Create a new sequence
3.  Add Footage to Timeline
4.  Select one or multiple clips
5.  Go to File Menu
6.  Select ‘Adobe Dynamic Link’
7.  Select ‘Replace with After Effects Composition’
8.  After Effects Opens
9.  Rename Comp in AE (optional)
10. Add Adjustment Layer
11. Add Tritone Effect Effect
12. Return To Premiere Pro
13. Locate new AE Comp in Project Bin
14. Rename (optional)
15. Return to AE to make changes
