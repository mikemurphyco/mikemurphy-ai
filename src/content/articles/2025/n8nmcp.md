🪜 **How To Install N8N-MCP Server in Claude Desktop:**

1.  Install Claude Desktop App
2.  Install Node.js
3.  Go To GitHub Repo
4.  Copy npx n8n-mcp (snippet)
5.  Open Terminal
6.  Paste in snippet & press Return
7.  Stop Terminal Process (Ctrl + C)
8.  Return To GitHub Repo & Copy Full Configuration File
9.  In Claude Desktop, open Settings
10. Click on 'Developer'
11. Click on 'Edit Config'
12. Add N8N URL & API Key
13. Save Config File
14. Quit Claude Desktop & Re-Open
15. Click on 'Settings & Tools' to confirm n8n-mcp is active
16. Return to GitHub Repo & Copy 🤖 Claude Project Setup
17. Create a New Project in Claude Desktop (Paid Plans only)
18. Click + Instructions and paste 🤖 Claude Project Setup
19. Enter Prompt in Claude to build n8n workflow and press 'Submit'
20. Accept permission requests
21. Copy JSON File when Claude is finished
22. Open N8N and click '+' Workflow
23. Paste JSON Code or click Ellipsis and 'Import File'
24. Set Credentials
25. Vibe Code with Claude to fix issues or make changes
26. Test n8n Workflow and activate when finished
27. Rinse & Repeat!
