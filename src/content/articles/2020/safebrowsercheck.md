### What is Google Chrome Safe Browser Check?

It is a new safety check feature in Google Chrome Web Browser ([**version 83.0 or later**](https://www.google.com/chrome/)) to run a quick test and make sure you are having a safe browsing experience while using Google Chrome. It takes a few seconds to run and a great new security feature to give you peace of mind.

#### The Safe Browser Check Includes:

1.  **Updates** Is Google Chrome is up to date?
2.  **Passwords** Do you have compromised passwords? If you do, the Safe Browser Check will show you how to fix them.
3.  **Safe Browsing** Is Safe Browsing turned on to protect you from malicious sites?
4.  **Extensions** Do you have any potentially harmful extensions installed? If so, the Safe Browser Check will tell you how to remove them.

> **Download Google Chrome Browser:** [https://www.google.com/chrome/](https://www.google.com/chrome/)

### How To Run Google Safety Check

1.  Open Google Chrome Web Browser
2.  Be sure you are signed into Google
3.  Click on 3 circles in the top right to open
4.  Click on Settings (or just use cmd + comma)
5.  Click ’Safety Check’ in the left sidebar
6.  Click the blue-button that says ‘Check Now’
7.  Review results and take action if required.

### How To Turn On Safe Browsing

This is to protect against dangerous websites, downloads, and extensions.

1.  Open Google Chrome Web Browser
2.  Be sure you are signed into Google
3.  Click on 3 circles in the top right to open
4.  Click on Settings
5.  Click ’Privacy & Security’ in the left sidebar
6.  Select ‘Enhanced’ or ‘Standard’ protection
