### 🪜**What You Will Learn:**

1.  Sign in to Hostinger VPS
2.  Click on Docker...Docker Catalog
3.  Search For OpenClaw & Click Deploy
4.  Copy OpenClaw Gateway Token in 1Password or secure note
5.  Generate LLM API Key (Anthropic, OpenAI, Gemini, Grok)
6.  Enter API Key in Config Form & click Deploy
7.  Look for Port Number in OpenClaw Docker Container
8.  Add Firewall Rule for OpenClaw Port Number (Accept, TCP, Port \#, Anywhere)
9.  Click Port Link to go to OpenClaw Gateway & Enter Token to Sign In
10. Chat with OpenClaw Bot to test connection
11. Add Messaging App (WhatsApp, etc)
12. Setup Name & Personality
13. Have fun!
