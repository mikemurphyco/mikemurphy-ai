**Description:**  
In this tutorial, you will learn how to use Claude Code to move Uptime Kuma Docker Container into its own directory, configure Custom Domains & Set Up Docker Network.

**What is Uptime Kuma?**  
A free, open-source monitoring tool that lets you monitor websites, APIs, and other services through a self-hosted dashboard.

**Prerequisites:**   
Claude Code installed  
SSH connection to Hostinger VPS  
Docker installed on VPS

### 🪜**What You Will Learn:**

1.  Launch Code Code using --resume
2.  Select Previous Claude Session & Tap Enter
3.  Press Shift + Tab Key (2x) to switch to Plan Mode
4.  Enter 3 Tasks for Claude Code (Move Uptime Kuma, Configure Custom Domain, Set up Docker Container)
5.  Review Claude Code Implementation Plan
6.  Make Edits or Accept and Proceed
7.  Login to Hostinger VPS Dashboard
8.  Go to DNS Manager and add A-Record pointing to VPS IP
9.  Test Custom Domain
10. Confirm Docker Network was created (app-net)
11. Update Docker Containers
